# Stretto Assembly

Stretto Assembly is a deterministic search system for contrapuntal stretto construction.

## Primary Capabilities

1. **Stretto Discovery**
   - Enumerates admissible delayed entries for a subject.
   - Applies rule-based overlap validation and dissonance constraints.
   - Ranks candidates for inspection and downstream composition.

2. **Stretto Chain Generation**
   - Composes pairwise-valid entries into multi-entry chains.
   - Enforces continuity constraints across successive delays and inversion choices.
   - Preserves consistent pivot semantics across entry expansion.

3. **Implied Harmony Detection (In Scope)**
   - Computes harmonic implications generated by stretto overlaps.
   - Supports rule-based inspection of consonance/dissonance behavior across chain states.
   - Treats harmonic interpretation as part of stretto evaluation, not as an unrelated diagnostic surface.

4. **Export (In Scope)**
   - **MIDI export** of assembled stretto with entries allocated by voice.
   - **ABC export** of accepted stretto results for symbolic score workflows.

## Input Model

- **Default input**: ABC notation entered directly in Stretto Lab.
- **Optional input**: MIDI import for interoperability.

The computational pipeline is tick-discrete after parsing. ABC is the canonical source because it encodes metrically exact symbolic durations, reducing ambiguity in overlap-window evaluation.

## Execution Flow

1. Enter or load ABC in Stretto Lab.
2. Configure search parameters (interval set, inversion policy, chain constraints).
3. Run discovery or chain search.
4. Inspect implied harmony and ranking outputs.
5. Export accepted results as MIDI and/or ABC.
